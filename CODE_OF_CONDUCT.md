# Code of Conduct

## Expected Behavior

People participating in this project are expected to:

- be respectful in issues, pull requests, reviews, and discussions,
- assume good intent while still being direct about technical concerns,
- focus criticism on ideas, code, and behavior rather than people,
- help keep discussions clear, constructive, and on topic.

## Unacceptable Behavior

This project does not accept:

- harassment, threats, or intimidation,
- hateful, discriminatory, or demeaning language,
- personal attacks or repeated hostile behavior,
- deliberate disruption of project discussions or review processes,
- publishing private information without consent.

## Scope

This code of conduct applies to project spaces such as issues, pull requests, reviews, discussions, and other repository-linked communication.

## Reporting

If you encounter behavior that violates this code of conduct, open a private channel with the maintainers when possible rather than escalating publicly.

For security-sensitive problems, follow [docs/SECURITY.md](docs/SECURITY.md).

## Enforcement

Project maintainers may remove comments, close discussions, or limit participation when behavior makes the project less safe or less productive for others.
